# README - Prompt dan Workflow

Tanggal dibuat: 2026-05-04

Folder `Promt/` menyimpan brief kerja, prompt AI, dan panduan revisi yang dipakai selama siklus pengembangan. Nama folder dipertahankan apa adanya untuk kompatibilitas dengan struktur yang sudah ada.

## Isi Folder

| File | Fungsi |
|------|--------|
| [COPILOT_BUGFIX_TASK.md](COPILOT_BUGFIX_TASK.md) | Brief tugas bugfix untuk AI assistant. |
| [COPILOT_PADAL_MEMBERS_CONNECT.md](COPILOT_PADAL_MEMBERS_CONNECT.md) | Prompt untuk fitur relasi Padal dan anggota. |
| [COPILOT_SIDEBAR_FIX.md](COPILOT_SIDEBAR_FIX.md) | Prompt/perintah terkait perbaikan sidebar. |
| [PANDUAN_REVISI_PROJECTPOLDA.md](PANDUAN_REVISI_PROJECTPOLDA.md) | Panduan teknis lengkap untuk siklus revisi project. |

## Rekomendasi Penggunaan

- Gunakan folder ini untuk bahan kerja dan jejak instruksi, bukan sebagai referensi perilaku sistem final.
- Jika prompt menghasilkan perubahan penting, ringkas hasilnya ke `After Revisi/` atau `01_System_Reference/`.
