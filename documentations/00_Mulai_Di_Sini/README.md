# Mulai Di Sini

Tanggal dibuat: 2026-05-04

Folder ini adalah titik masuk tercepat untuk memahami dokumentasi ProjectPolda tanpa harus membaca seluruh arsip sekaligus.

## Urutan Baca Rekomendasi

1. Baca [../README.md](../README.md) untuk peta besar dokumentasi.
2. Baca [../01_System_Reference/README.md](../01_System_Reference/README.md) untuk memahami alur dan fitur sistem.
3. Baca [../After Revisi/README.md](../After%20Revisi/README.md) untuk kondisi implementasi terbaru.
4. Baca [../07_Remediation_Program/README.md](../07_Remediation_Program/README.md) untuk konteks bug, akar masalah, dan perbaikannya.
5. Gunakan [STANDAR_DOKUMENTASI.md](STANDAR_DOKUMENTASI.md) sebelum menambah atau memperbarui dokumen.

## Untuk Siapa Folder Ini

- Developer baru yang butuh onboarding cepat.
- Reviewer yang ingin tahu perubahan terbaru tanpa membaca arsip penuh.
- Maintainer yang ingin menjaga konsistensi format dokumentasi.

## Prinsip Penggunaan

- Gunakan folder ini sebagai panduan masuk, bukan tempat menyimpan semua dokumen baru.
- Simpan dokumen baru di area yang sesuai, lalu tambahkan link-nya ke indeks yang relevan.
- Jika sebuah perubahan mengubah perilaku sistem, update minimal satu dokumen referensi dan satu dokumen histori revisi.
