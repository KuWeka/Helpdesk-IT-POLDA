# README - Before Revisi

Tanggal dibuat: 2026-05-04

Folder ini menyimpan arsip dokumentasi sebelum rangkaian revisi terbaru dilakukan. Kontennya dipertahankan sebagai referensi historis dan bahan pembanding.

## Karakter Folder

- Bersifat historis.
- Tidak menjadi sumber utama kondisi sistem saat ini.
- Tetap penting untuk audit keputusan, jejak perubahan, dan pembandingan arsitektur lama vs baru.

## Subfolder

| Folder | Fokus |
|--------|-------|
| `01_System_Reference/` | Referensi sistem lama |
| `02_API_Documentation/` | Dokumentasi API dan ringkasan refactor lama |
| `03_Web_UI_Transformation/` | Rencana dan eksekusi transformasi UI |
| `04_Backend_Operational_Guides/` | Panduan operasional backend |
| `05_Release_and_Versioning/` | Release notes, versioning, dan fitur sistem |
| `06_Delivery_Execution_Records/` | Catatan eksekusi delivery dan phase logs |
| `07_Remediation_Program/` | Program remediasi versi lama |
| `08_AI_Prompts_and_Workflows/` | Prompt dan workflow AI historis |
| `09_Legacy_Architecture/` | Arsip arsitektur lama |
| `10_Archive/` | Arsip laporan dan dokumen lama |

## Aturan Pemeliharaan

- Hindari perubahan isi substansial kecuali untuk metadata atau indeks.
- Jika ada dokumen lama yang perlu dirujuk, link-kan dari dokumen baru alih-alih menyalin ulang isi.
